HTML로 다양한 내용 입력하기
--
```
- <hn> 태그 - 제목 표시
    - 기본형: <hn>제목</hn>
    - 크기: h1>h2>h3>h4>h5

- <p> 태그 - 텍스트 단락
    - 기본형: <p>텍스트</p>
    - 입력한 내용 앞뒤로 빈 줄이 생기면서 텍스트 단락이 만들어짐

- <br> 태그 - 줄 바꾸기
    - 기본형: <br>
    - 줄을 바꿀 위치에 <br> 태그를 사용. 닫는 태그가 없음

- <hr> 태그 - 분위기 전환
    - 기본형: <hr>
    - 가로줄 표시되지만 CSS를 사용해 가로줄 감출 수 있음

- <strong>, <b> 태그 - 굵게 표시
    - 기본형: <strong>내용</strong>, <b>내용</b>
    - <strong>: 중요한 내용이라서 강조해야 할 때
    - <b>: 단순히 굵게 표시할 때

- <em>, <i>, <cite> - 이탤릭체로 표시하기
    - 기본형: <em>내용</em>, <i>내용</i>, <cite>저작물 제목</cite>
    - <em>: 흐름상 특정 부분을 강조하고 싶을 때
    - <i>: 단순히 이탤릭체로 표시할 때
    - <cite>: 책이나 영화, 블로그 같은 저작물의 제목 표시

- <blockquote> 태그 - 인용문 넣기
    - 기본형: <blockqutoe>인용 내용</blockquote>
    - 다른 텍스트보다 안으로 들여써짐

- <ol>, <li> - 순서 목록
    - 기본형: <ol><li>항목1</li></ol>
    - 각 항복 앞에 숫자가 붙여짐

- <ul>, <li> - 순서 없는 목록
    - <ul><li>항목1</li></ul>
    - 각 항목 앞에 블릿이 붙여짐

- 테이블
    - <table>: 표 전체
    - <caption>: 표 제목
    - <tr>: 행
    - <td>: 셀
    - <th>: 제목 셀
