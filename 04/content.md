HTML로 다양한 내용 입력하기
--
```
- <hn> 태그 - 제목 표시
    - 기본형: <hn>제목</hn>
    - 크기: h1>h2>h3>h4>h5

- <p> 태그 - 텍스트 단락
    - 기본형: <p>텍스트</p>
    - 입력한 내용 앞뒤로 빈 줄이 생기면서 텍스트 단락이 만들어짐

- <br> 태그 - 줄 바꾸기
    - 기본형: <br>
    - 줄을 바꿀 위치에 <br> 태그를 사용. 닫는 태그가 없음

- <hr> 태그 - 분위기 전환
    - 기본형: <hr>
    - 가로줄 표시되지만 CSS를 사용해 가로줄 감출 수 있음

- <strong>, <b> 태그 - 굵게 표시
    - 기본형: <strong>내용</strong>, <b>내용</b>
    - <strong>: 중요한 내용이라서 강조해야 할 때
    - <b>: 단순히 굵게 표시할 때

- <em>, <i>, <cite> - 이탤릭체로 표시하기
    - 기본형: <em>내용</em>, <i>내용</i>, <cite>저작물 제목</cite>
    - <em>: 흐름상 특정 부분을 강조하고 싶을 때
    - <i>: 단순히 이탤릭체로 표시할 때
    - <cite>: 책이나 영화, 블로그 같은 저작물의 제목 표시

- <blockquote> 태그 - 인용문 넣기
    - 기본형: <blockqutoe>인용 내용</blockquote>
    - 다른 텍스트보다 안으로 들여써짐

- <ol>, <li> - 순서 목록
    - 기본형: <ol><li>항목1</li></ol>
    - 각 항복 앞에 숫자가 붙여짐

- <ul>, <li> - 순서 없는 목록
    - <ul><li>항목1</li></ul>
    - 각 항목 앞에 블릿이 붙여짐

- 테이블
    - <table>: 표 전체
    - <caption>: 표 제목
    - <tr>: 행
    - <td>: 셀
    - <th>: 제목 셀

- 이미지
    - <img> 태그의 src 속성
        - 웹 문서 파일의 위치를 기준으로 이미지 경로 지정
        - 웹 문서 안에 삽입하려는 이미지 파일을 작업 폴더 안에 가지고 있거나 인터넷에 올라가 있어야 함
        - HTML 과 이미지 파일이 같은 폴더에 있다면 src 속성에 파일 이름만 적음
        - 다른 곳에 이미지 파일이 있다면 폴더 경로까지 같이 적용
    - <img> 태그의 alt 속성
        - 이미지를 설명하는 대체 텍스트
        - 화면 낭독기에서 이미지 대신 대체 텍스트를 읽어 줌
        - 블릿 이미지나 작은 아이콘 등의 경우는 alt=""

- 오디오와 비디오 태그
    - <audio> 태그
        - <audio src="오디오 파일 경로"></audio>
    - <video> 태그
        - <video src="비디오 파일 경로"></video>
    - 속성
        - controls: 플레이어 화면에 컨트롤 바를 표시함
        - autoplay: 오디오나 비디오를 자동으로 실행함
        - loop: 오디오나 비디오를 반복 재생함
        - muted: 오디오나 비디오의 소리를 제거함
        - preload: 페이지를 불러올 때 오디오나 비디오 파일을 어떻게 로딩할 것인지를 지정함
        - width, height: 웹 문서 상의 비디오 화면의 크기로, 비디오 플레이어의 너비와 높이를 지정함
        - poster: 비디오가 재생되기 전까지 화면에 표시될 포스터 이미지를 지정함

- 하이퍼링크 삽입하기
    - <a> 태그
        - 다른 문서, 혹은 다른 사이트로 바로 연결해주는 기능
        - 기본형: <a href="링크할 주소">텍스트 또는 이미지 </a>
        - target 속성: blank 로 지정하면 연결된 문서가 새 탭으로 열림
